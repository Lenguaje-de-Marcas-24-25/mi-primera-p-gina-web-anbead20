Ejemplos de Listas y Términos HTML

Términos Generales

W3C:
Consorcio que desarrolla estándares para la Web. Ejemplo de una lista de estándares del W3C:
  • HTML
  • CSS
  • JavaScript
WHATWG:
Grupo de trabajo que promueve la evolución de HTML y tecnologías afines. A continuación, una lista ordenada de estándares según el orden en que fueron adoptados:
  1. HTML5
  2. Canvas
  3. DOM Manipulation
HTML:
Lenguaje estándar de marcado para la creación de páginas web. Ejemplo de una lista anidada de elementos HTML:
  • HTML
    • Encabezados (h1-h6)
    • Párrafos (p)
    • Enlaces (a)
XHTML:
Extensión de HTML que sigue las reglas de XML, utilizada para crear estructuras de páginas organizadas. Ejemplo de un menú de un sitio XHTML:
  • Inicio
  • Noticias
  • Contacto

    • Elementos válidos

    • Entidades

    APIs en HTML

    Drag and Drop:
    API que permite arrastrar y soltar elementos en una página web, útil para listas interactivas.
    Geolocalización:
    API que permite obtener la ubicación geográfica del usuario, ideal para menús de navegación basados en ubicación.
    Almacenamiento local:
    API para almacenar datos en el navegador de manera persistente. Útil para guardar preferencias de listas ordenadas o sin orden del usuario.
    App Cache:
    API que permite almacenar recursos en el navegador para acceso sin conexión, asegurando el acceso a listas y menús offline.
    WebWorkers:
    Permite ejecutar scripts en segundo plano, manteniendo listas o tareas actualizadas sin afectar la navegación del usuario.
    Editor WYSIWYG:
    Editor visual que permite ver el contenido como se mostrará al usuario final, útil para crear y editar listas y menús en HTML.

    Navegadores Compatibles

    Mozilla Firefox:
    Navegador de código abierto desarrollado por Mozilla, compatible con todas las listas HTML y APIs de HTML5.
    Google Chrome:
    Navegador desarrollado por Google, ampliamente compatible con listas HTML y APIs avanzadas.
    Safari:
    Navegador de Apple para dispositivos macOS y iOS, compatible con XHTML y listas anidadas.
    Opera:
    Navegador con funciones avanzadas, como bloqueo de anuncios, y soporte completo para listas y APIs de HTML5.
